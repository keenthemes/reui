ReUI, as a first-class shadcn registry, is evolving to contribute meaningfully to the shadcn ecosystem.

Our focus is on providing exceptional [Components](/components) that are both
production-ready and AI-ready, empowering developers to ship their shadcn projects faster and smarter.

## Current Status

- **ReUI 2.0 release**
- **10+ new components**
- **1000+ component examples**

## What's Next

- Expand the **Components** set with new unique primitives.
- Grow the **Components** collection for more use cases.
- Deepen Base UI and Radix UI coverage across every primitive.
