## API Reference

This section provides a comprehensive overview of the data grid components, detailing their custom props and linking to relevant documentation.

### DataGrid

This component is based on the [DataGrid](https://tanstack.com/table/latest/docs/guide/data) from [TanStack Table](https://tanstack.com/table/latest) and includes the following custom props:

| **Prop**                                                                                           | **Type**                                                    | **Default**           |
| -------------------------------------------------------------------------------------------------- | ----------------------------------------------------------- | --------------------- |
| `className` <PopoverInfo>Additional class names for styling the data grid container.</PopoverInfo> | ~string~                                                    | <Na/>                 |
| `table` <PopoverInfo>The TanStack Table instance to be used by the data grid.</PopoverInfo>        | ~Table\<TData>~                                             | <Na/>                 |
| `recordCount` <PopoverInfo>Total number of records in the data grid.</PopoverInfo>                 | ~number~                                                    | <Na/>                 |
| `children` <PopoverInfo>Child components to be rendered within the data grid.</PopoverInfo>        | ~ReactNode~                                                 | <Na/>                 |
| `onRowClick` <PopoverInfo>Callback function triggered when a row is clicked.</PopoverInfo>         | ~(row: TData) => void~                                      | <Na/>                 |
| `isLoading` <PopoverInfo>Indicates if the data grid is in a loading state.</PopoverInfo>           | ~boolean~                                                   | ~false~               |
| `loadingMode` <PopoverInfo>Specifies the loading mode for the data grid.</PopoverInfo>             | ~enum~ <PopoverInfo> "skeleton" \| "spinner" </PopoverInfo> | ~skeleton~            |
| `loadingMessage` <PopoverInfo>Message displayed during loading state.</PopoverInfo>                | ~ReactNode \| string~                                       | <Na/>                 |
| `emptyMessage` <PopoverInfo>Message displayed when there are no records.</PopoverInfo>             | ~ReactNode \| string~                                       | ~"No data available"~ |
| `tableLayout` <PopoverInfo>Configuration for the table layout.</PopoverInfo>                       | ~object~                                                    | See below             |
| `tableClassNames` <PopoverInfo>Custom class names for table elements.</PopoverInfo>                | ~object~                                                    | See below             |

#### Table Layout Props - `tableLayout`

| **Prop**                                                                           | **Type**            | **Default** |
| ---------------------------------------------------------------------------------- | ------------------- | ----------- |
| `dense` <PopoverInfo>Controls compact table layout.</PopoverInfo>                  | ~boolean~           | ~false~     |
| `cellBorder` <PopoverInfo>Enables borders between cells.</PopoverInfo>             | ~boolean~           | ~false~     |
| `rowBorder` <PopoverInfo>Enables borders between rows.</PopoverInfo>               | ~boolean~           | ~true~      |
| `rowRounded` <PopoverInfo>Applies rounded corners to rows.</PopoverInfo>           | ~boolean~           | ~false~     |
| `stripped` <PopoverInfo>Enables striped rows.</PopoverInfo>                        | ~boolean~           | ~false~     |
| `headerBackground` <PopoverInfo>Shows header background.</PopoverInfo>             | ~boolean~           | ~true~      |
| `headerBorder` <PopoverInfo>Shows header border.</PopoverInfo>                     | ~boolean~           | ~true~      |
| `headerSticky` <PopoverInfo>Makes header sticky.</PopoverInfo>                     | ~boolean~           | ~false~     |
| `width` <PopoverInfo>Sets table width mode.</PopoverInfo>                          | ~'auto' \| 'fixed'~ | ~'fixed'~   |
| `layout` <PopoverInfo>Sets table width mode.</PopoverInfo>                         | object              | ~'fixed'~   |
| `columnsVisibility` <PopoverInfo>Enables column visibility controls.</PopoverInfo> | ~boolean~           | ~false~     |
| `columnsResizable` <PopoverInfo>Enables column resizing.</PopoverInfo>             | ~boolean~           | ~false~     |
| `columnsPinnable` <PopoverInfo>Enables column pinning.</PopoverInfo>               | ~boolean~           | ~false~     |
| `columnsMovable` <PopoverInfo>Enables column moving.</PopoverInfo>                 | ~boolean~           | ~false~     |
| `columnsDraggable` <PopoverInfo>Enables column drag & drop.</PopoverInfo>          | ~boolean~           | ~false~     |
| `rowsDraggable` <PopoverInfo>Enables row drag & drop.</PopoverInfo>                | ~boolean~           | ~false~     |

#### Table Class Names - `tableClassNames`

| **Prop**                                                           | **Type** | **Default**                                             |
| ------------------------------------------------------------------ | -------- | ------------------------------------------------------- |
| `base` <PopoverInfo>Base class for the table.</PopoverInfo>        | ~string~ | <Na/>                                                   |
| `header` <PopoverInfo>Class for the header section.</PopoverInfo>  | ~string~ | <Na/>                                                   |
| `headerRow` <PopoverInfo>Class for header rows.</PopoverInfo>      | ~string~ | <Na/>                                                   |
| `headerSticky` <PopoverInfo>Class for sticky header.</PopoverInfo> | ~string~ | ~"sticky top-0 z-10 bg-background/90 backdrop-blur-xs"~ |
| `body` <PopoverInfo>Class for the table body.</PopoverInfo>        | ~string~ | <Na/>                                                   |
| `bodyRow` <PopoverInfo>Class for body rows.</PopoverInfo>          | ~string~ | <Na/>                                                   |
| `footer` <PopoverInfo>Class for the footer section.</PopoverInfo>  | ~string~ | <Na/>                                                   |
| `edgeCell` <PopoverInfo>Class for edge cells.</PopoverInfo>        | ~string~ | <Na/>                                                   |

### DataGridColumnFilter

This component provides filtering capabilities for data grid columns. It utilizes [TanStack Table Columns](https://tanstack.com/table/latest/docs/guide/columns) for column management and filtering.

| **Prop**                                                                       | **Type**                                                                                                                               | **Default** |
| ------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------- | ----------- |
| `column` <PopoverInfo>The column to which the filter is applied.</PopoverInfo> | ~Column\<TData, TValue>~                                                                                                               | <Na/>       |
| `title` <PopoverInfo>Title of the filter popover.</PopoverInfo>                | ~string~                                                                                                                               | <Na/>       |
| `options` <PopoverInfo>Filter options available for the column.</PopoverInfo>  | ~Array~ <PopoverInfo> Array\<\{ label: string; value: string; icon?: React.ComponentType\<\{ className?: string \}> \}> </PopoverInfo> | <Na/>       |

### DataGridColumnHeader

This component provides a header for data grid columns, including sorting and pinning controls. It is built on top of [TanStack Table Header](https://tanstack.com/table/latest/docs/guide/headers) for efficient data management.

| **Prop**                                                                          | **Type**                 | **Default** |
| --------------------------------------------------------------------------------- | ------------------------ | ----------- |
| `column` <PopoverInfo>The column for which the header is displayed.</PopoverInfo> | ~Column\<TData, TValue>~ | <Na/>       |
| `title` <PopoverInfo>Title of the column header.</PopoverInfo>                    | ~string~                 | <Na/>       |
| `icon` <PopoverInfo>Icon displayed in the column header.</PopoverInfo>            | ~ReactNode~              | <Na/>       |
| `filter` <PopoverInfo>Filter component for the column header.</PopoverInfo>       | ~ReactNode~              | <Na/>       |

### DataGridColumnVisibility

This component provides controls for toggling the visibility of data grid columns. It utilizes [TanStack Table Column Visibility](https://tanstack.com/table/latest/docs/guide/column-visibility) for managing column visibility.

| **Prop**                                                                                            | **Type**        | **Default** |
| --------------------------------------------------------------------------------------------------- | --------------- | ----------- |
| `table` <PopoverInfo>The table instance to which the visibility controls are applied.</PopoverInfo> | ~Table\<TData>~ | <Na/>       |
| `trigger` <PopoverInfo>The trigger element for the visibility dropdown.</PopoverInfo>               | ~ReactNode~     | <Na/>       |

### DataGridPagination

This component provides pagination controls for the data grid. It is built on top of [TanStack Table Pagination](https://tanstack.com/table/latest/docs/api/features/pagination) for efficient data management.

| **Prop**                                                                                                   | **Type**    | **Default**                     |
| ---------------------------------------------------------------------------------------------------------- | ----------- | ------------------------------- |
| `sizes` <PopoverInfo>Array of page sizes available for selection.</PopoverInfo>                            | ~number[]~  | ~[5, 10, 25, 50, 100]~          |
| `sizesLabel` <PopoverInfo>Label for the page size selector.</PopoverInfo>                                  | ~string~    | ~"Show"~                        |
| `sizesDescription` <PopoverInfo>Description text for the page size selector.</PopoverInfo>                 | ~string~    | ~"per page"~                    |
| `sizesInfo` <PopoverInfo>Additional info text for the page size selector.</PopoverInfo>                    | ~string~    | <Na/>                           |
| `sizesSkeleton` <PopoverInfo>Skeleton component to display while loading page size selector.</PopoverInfo> | ~ReactNode~ | <Na/>                           |
| `more` <PopoverInfo>Enables pagination grouping with ellipsis.</PopoverInfo>                               | ~boolean~   | ~false~                         |
| `moreLimit` <PopoverInfo>Number of page buttons to show before using ellipsis.</PopoverInfo>               | ~number~    | ~5~                             |
| `info` <PopoverInfo>Template for displaying pagination information.</PopoverInfo>                          | ~string~    | ~"\{from} - \{to} of \{count}"~ |
| `infoSkeleton` <PopoverInfo>Skeleton component to display while loading pagination info.</PopoverInfo>     | ~ReactNode~ | <Na/>                           |
| `className` <PopoverInfo>Additional class names for styling the pagination container.</PopoverInfo>        | ~string~    | <Na/>                           |
| `rowsPerPageLabel` <PopoverInfo>Label text for the rows per page selector.</PopoverInfo>                   | ~string~    | ~"Rows per page"~               |
| `previousPageLabel` <PopoverInfo>Screen reader text for the previous page button.</PopoverInfo>            | ~string~    | ~"Go to previous page"~         |
| `nextPageLabel` <PopoverInfo>Screen reader text for the next page button.</PopoverInfo>                    | ~string~    | ~"Go to next page"~             |
| `ellipsisText` <PopoverInfo>Text displayed in ellipsis buttons for pagination.</PopoverInfo>               | ~string~    | ~"..."~                         |

### DataGridTableDnd

This component provides drag-and-drop functionality for data grid rows and columns using [Dnd Kit](https://dndkit.com/).

| **Prop**                                                                                          | **Type**                        | **Default** |
| ------------------------------------------------------------------------------------------------- | ------------------------------- | ----------- |
| `handleDragEnd` <PopoverInfo>Callback function when a drag-and-drop operation ends.</PopoverInfo> | ~(event: DragEndEvent) => void~ | <Na/>       |

### DataGridTable

This component provides the table structure for the data grid, including headers, rows, and cells. It is built on top of [TanStack Table](https://tanstack.com/table/latest/docs/guide/tables) for efficient data management.

| **Prop**                                                                                | **Type**    | **Default** |
| --------------------------------------------------------------------------------------- | ----------- | ----------- |
| `children` <PopoverInfo>Child components to be rendered within the table.</PopoverInfo> | ~ReactNode~ | <Na/>       |

### ColumnMeta Interface

This interface is used to define custom metadata for columns in the data grid. It is built on top of [ColumnMeta](https://tanstack.com/table/latest/docs/api/core/column-def#meta).

| **Prop**                                                                                    | **Type**                    | **Default** |
| ------------------------------------------------------------------------------------------- | --------------------------- | ----------- |
| `headerTitle` <PopoverInfo>Custom title for the column header.</PopoverInfo>                | ~string~                    | <Na/>       |
| `headerClassName` <PopoverInfo>Additional class names for the column header.</PopoverInfo>  | ~string~                    | <Na/>       |
| `cellClassName` <PopoverInfo>Additional class names for the column cells.</PopoverInfo>     | ~string~                    | <Na/>       |
| `skeleton` <PopoverInfo>Skeleton component to display while loading data.</PopoverInfo>     | ~ReactNode~                 | <Na/>       |
| `expandedContent` <PopoverInfo>Function to render expanded content for a row.</PopoverInfo> | ~(row: TData) => ReactNode~ | <Na/>       |
